##КАК ПОЛЬЗОВАТЬСЯ##
Данное расширение это ДОПОЛНЕНИЕ к уже существующему встроенному расширение Image Generation
Сначала настройте его, а после пользуйтесь этим расширением

В поле "Инструкция для генерации описания" вы пишите подсказки для вашего ИИ, по умолчанию уже стоит подсказка которая хорошо работает, но если что вы можете изменить её
Кнопка "Сгенерировать описание"  делает одноимённое действие. Оно описывает внешний вид персонажа, с которым вы играете (исходя из подсказки). Но иногда ИИ может выдать не верное описание персонажа, или выдать описание вас, а не персонажа с которым вы играете, поэтому, при завершении описания, вам предоставляется выбор между текущим описанием и тем что сгенерировалось только что. 
Оба блока полностью редактируемы. Кнопка "Заполнить поле и сгенерировать изображение" вставляет описание персонажа которое было сгенерировано в Character-specific prompt prefix тем самым последущая генерация изображения не добавит лишних и не нужных деталей.
